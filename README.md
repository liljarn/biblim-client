# Клиентская часть библиотечной системы - Customerfront

## Описание проекта

Фронтенд-приложение для библиотечной системы, предоставляющее пользовательский интерфейс для взаимодействия с публичным API. Система позволяет пользователям просматривать каталог книг, регистрироваться, управлять своим профилем, бронировать книги и оставлять комментарии.

## Технологический стек

### Основные технологии
- **Язык программирования**: TypeScript
- **Фреймворк**: React 18
- **Сборщик**: Vite
- **UI библиотека**: Gravity UI Kit

### Управление состоянием и формы
- **State Management**: Redux Toolkit
- **Формы**: React Hook Form
- **Валидация**: Yup

## Инструкция для запуска

### Переменные окружения
Перед запуском убедитесь, что в файле `src/shared/config/consts.ts` указаны правильные URL для backend-сервисов.

Для системы есть 2 процесса запуска: **локальный запуск** и **запуск в docker-контейнере**.

### Локальный запуск

1. Установите зависимости:
```bash
npm install
```
2. Запустите development сервер:
```bash
npm run dev
```
3. Приложение будет доступно по адресу: http://localhost:5173

### Запуск в Docker

1. Убедитесь, что в файле `src/shared/config/consts.ts` указаны правильные hostname для backend-сервисов

2. Запустите контейнер:
```bash
docker compose up -d
```
